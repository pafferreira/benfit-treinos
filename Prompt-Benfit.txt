-- Prompt para Meus Treinos - BENFIT
-- 0 1/12/2025 - Paulo Affonso - Revisão: 14/12/2025

Vamos criar um aplicativo de acompanhamento de treinos e exercícios físicos
Possui um treinador chamado Benfit, um avatar de um rapaz simpático, jovem, cabelo escuro, olhos grandes, forte com os músculos definidos mas sem exagero.
Este treinador será um agente de IA que conhece muito de exercícios físicos, entende de fisiologia e sabe dosar os treinos para cada tipo de usuário, homens, mulheres, velho, jovem, e definir os objetivos dos exercícios. Tem um leve toque de humor e inteligência.
Este aplicativo vai incluir conquistas de medalhas e trofeus como gamificação. 
Pensei num formato de gamificação para calcular pontuação e premiar com imagens de medalhas e trofeus e frases de incentivo
Gamificação
  +50 pontos por sessão concluída
  níveis (Bronze, Prata, Ouro, Diamante), troféu e frases de incentivo

Procure um modelo de aplicativo com UI/UX das melhores academias globais.
Vamos apresentar um visual agradável com cores claras, utilizando avatares para representar os usuários.
Procure alguns Avatars com imagem 3D , simpaticos e levemente engraçados.

O aplicativo deve ser para mobile e apresentar interface de fácil navegação, otimizando espaço visual tendo:
Tela inicial 
- identificação do usuário com nome, possibilidade de criar novo usuário
- quantidade de sessões de exercícios ex: 05/30
- dashboard resumido com quantidade de exercícios, metas, medalhas e trofeus conquistados
- Apresentação do Benfit como seu treinador , pode incluir um avatar no formato de desenho 3D
- Cada usuário pode escolher entre 10 avatares para seu perfil com  imagens de avatares entre homens e mulheres.

Botões de Criar Novo Treino , Meu Treino
Tela Criar Novo Treino
- Benfit vai criar uma sessão de treinos perguntando, Objetivo, Grupo de Músculos, frequência de dias por semana
- O Benfit deve criar e manter listas de exercício previamente em umbanco de dados
- Vai apresentar as sessões de exercícios remcomendadas dentro de um mês 

Tela de Meu Treino
- Apresenta a sessões criadas para o usuários, como Treino A, Treino B, Treino C
- Após escolher o Treino abre o accordion com a lista de exercícios com Chechbox de feito, imagem do exercício e/ou aparelho, Nome do exercício, repetições e dicas
- Quando executar um exercício, apresentar um pequeno cronômetro para medir o tempo de exercício e/ou descanso

Apresenta o seguinte escopo:
#1 Tela inicial (imagem de fundo de academia com aparelhos e pesos)
Tela de Boas-Vindas: Transforme-se com qualidade de Vida. Planos personalizados e acompanhamento de progresso para atingir seus objetivos.

#2 Tela de Registro/Login: 
  - - Permite que novos usuários criem uma conta ou que usuários existentes façam login.
  - selecionar/criar usuário

#3 Tela Dashboard
  - Nome do usuário com seu avatar
  - progresso mensal “sessões”: 05/30
  - Treinos da semana
  - dashboard (sessões completadas, plano atual, Medalhas e/ou Trofeu)
  - Meta(s) definida(s) em Perfil - se autorizado
  - apresenta o avatar do Benfit (AI Coach) um círculo flutuando no canto inferior direito 

#4 Planos de Treino
  - Apresenta os planos de treino executando, executados
  - Ao escolher um plano em excução ou executado abre a tela de Meu Treino
  - Pode criar seu próprio plano (leva para tela Criar Novo Treino)
  - Pode pedir para o Benfit sugerir, leva à tela do Benfit Coach 

#5 Tela Criar Novo Treino
  - Apresenta Treinos existentes que o usuário pode escolher
  - Benfit pode perguntar se deseja ajuda e questiona sobre Objetivo, Grupos de músculos (multi), Frequência (dias/semana)
  - A tela apresenta filtro por categoria (iniciante, intermediário, avançado) e busca por grupo de músculos (abdomen, bíceps, cardio, etc) , aparelho, músculo, isométrico, tapete
  - Ao escolher o Treino o programa gera 3–4 sessões (A/B/C/D) automáticas para 1 mês, com exercícios a partir de uma biblioteca de exercícios, que contém exercícios, descrições e imagens de aparelhos/pesos criada pelo Admin
  - Usuário pode substituir cada exercício por outro de uma lista da biblioteca considerando o mesmo grupo muscular
  - Informa os Sets (Séries) , Reps (Repetições) e Rest (Tempo de Descanso)
  - As Instruções vem da biblioteca de exercícios
  - No campo Notas podem ser escrito os detalhes
  - Para esta tela deve ser criada uma interface clara, que destaque bem para uma boa experiência do usuário, que pode ser um "accodion" apresentando o dia do exercício, abrir todos os exercício do dia, ao selecionar o exercício abri os detalhes como Set, Rep, Rest, outros.

#6 Tela Meu Treino
  - Mostra um pequeno calendário com destaque para os dias que já realizou exercícios
  - Mostrar o próximo treino pela a sequência do plano
  - Ao escolher o exercício da lista, utilizar uma tela especícifca para utilizar todo o espaço da tela onde apresenta os detalhes: Set, Rep, Rest e dicas
  - Apresentar um pequeno cronômetro para controlar o tempo de descanso (com intervalos sugeridos, ex. 60s , 90s)
  - Apresentar imagens de aparelhos ou do exercício
  - Apresentar um checkbox de "Feito" no card, quando o check é marcado, o sistema alimenta os Logs, Log do Dia e Log dos Exercícios

#7 Tela Biblioteca de Exercícios
	- Telas semelhante à tela específica de exercício de Meu Treino, porem com possibilidade de edição de todos os campos pelo Admin 
	- Apresentar uma barra de busca, filtros de músculos e equipamento
	- Permite criar novos exercícios
	- Cada exercício pode ter imagens dos músculos ou da maquina ou equipamento, pode ser ícones ou avatares buscados na internet

#8 Perfil do Usuário
  - Registrar informações do usuário (nome, email, peso, altura)
  - 10+ avatares para escolher (homens e mulheres)
  - Metas a alcançar (peso, caminhada ou distância 10K-20K, Flexibilidade e Foça, Bem-estar com yoga, alongamento, resistência (pode escolher mais de uma)
